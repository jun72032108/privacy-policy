1. 수집하는 정보

번역렌즈는 어떠한 개인정보도 수집하지 않습니다.

  • 서버로 전송되는 개인 데이터 없음
  • 계정 생성 또는 로그인 없음
  • 개인 식별 정보 수집 없음
  • 언어 설정(출발어/도착어)은 앱 편의를 위해 기기 내부에만 저장되며, 외부로 전송되지 않습니다

2. 화면 캡처 및 OCR 처리

본 앱은 번역 기능을 위해 기기 화면의 일부를 캡처합니다.

  • 캡처된 이미지는 기기 내부에서만 처리됩니다
  • 캡처 데이터는 외부 서버로 전송되지 않습니다
  • 캡처 이미지는 저장되지 않으며 처리 즉시 메모리에서 삭제됩니다
  • 텍스트 인식(OCR) 및 번역은 Google ML Kit을 통해 온디바이스로 처리됩니다
  • 비밀번호, 카드번호 등 민감한 정보가 화면에 있을 때는 번역 버튼 사용을 피하도록 권장합니다

4. 번역 모델 다운로드

본 앱은 Google ML Kit 번역 기능을 사용합니다. 앱 최초 실행 시 또는 새로운 언어 쌍 선택 시, 번역에 필요한 언어 모델 파일을 Google 서버에서 기기로 다운로드합니다.

  • 다운로드되는 것은 번역 모델 파일뿐이며, 사용자의 개인정보는 전송되지 않습니다
  • 다운로드된 모델은 기기에 저장되어 이후 오프라인에서도 사용 가능합니다
  • 모델 다운로드는 Google의 서버 정책을 따릅니다

5. 제3자 서비스

본 앱은 Google ML Kit을 사용합니다. 텍스트 인식(OCR) 및 번역 처리는 기기 내부에서만 이루어지나, 번역 모델 다운로드 시 Google 서버와 통신이 발생할 수 있습니다.

Google 개인정보처리방침: https://policies.google.com/privacy

6. 데이터 보관 기간

번역렌즈는 사용자 데이터를 서버에 보관하지 않습니다.

  • 화면 캡처 이미지: 번역 완료 즉시 메모리에서 삭제
  • 언어 설정: 앱 삭제 시 기기에서 자동 삭제
